# 跨平台簡報瑞士刀小工具

設計一個跨平台的簡報工具，執行後只會顯示在系統匣，以雷射筆光點為圖示視覺主題，在圖示上按滑鼠右鍵會出現選單。具備以下項目：

1. 聚光燈滑鼠游標：按一下展開設定面板，具有以下項目：
    - 啟用：勾選後會把游標改成大的半透明圓形聚光燈樣式，方便簡報時讓觀眾看到講師指的位置
    - 半透明顏色與透明度設定
    - 聚光燈圓圈大小設定
    - Windows 上須使用真正的系統游標，效果不可被一般視窗、工作列或系統匣遮擋；停用、結束或程式異常終止時必須恢復使用者原本的游標
2. 按鍵顯示：按一下展開設定面板，具有以下項目：
    - 啟用：勾選後會在螢幕中央下方以半透明圓角矩形顯示使用者按的組合鍵，不會吃掉按鍵，仍然要把按鍵往焦點視窗送。
    - 半透明顏色與透明度設定
3. 結束：取消上面各項工具的效果後結束程式
